update_transaction(db, transaction_id, amount)[source]¶

Check if the transaction exists in the database, and if so then store the amount into that transaction’s database entry.

Parameters:
  • db – database connection
  • transaction_id – the id of the transaction the user wants to update the amount of
  • amount – the new amount that the user wants to store into the transaction
Returns:

a string saying that the transaction does not exist, or if it does exist then a string saying ‘updated transaction’
